課題1: MulmoCast Context管理

動画生成時に「studioBeat.duration is not set」エラーが発生

❌ 失敗したコード

// 音声生成
await audio(context);
// 動画生成
await movie(context);
// ⚠️ エラー！duration未設定

問題：audio()の戻り値を捨てている

✅ 正しいコード

// 音声生成して戻り値を保存
const audioResult = await audio(context);
context = audioResult;
// 更新されたcontextで動画生成
await movie(context);
// ✅ 成功！durationあり

解決：audio()が返すcontextにdurationが含まれる
